# Minitest::Coloring
Coloring result of running minitest.

## Installation

Add a line below to your Gemfile.

    gem 'minitest-coloring'

And then run this.

    $ bundle install

## Usage

Add `require "minitest/coloring"` to your test file or pass `--color` as an argument while running your tests.
